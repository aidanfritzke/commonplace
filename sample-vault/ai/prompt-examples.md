# Reusable prompts

**Summarize a note**
> Summarize this in 3 sentences, preserving the key ideas. Plain prose, no preamble.

**Extract action items**
> List every action item as "- [ ] owner — task". If there are none, say "none".

**Rewrite for clarity**
> Rewrite this to be clearer and tighter while keeping my voice. Output only the rewrite.

**Rubber-duck a decision**
> Argue both sides in two short paragraphs, then give your honest recommendation.

**Name things**
> Suggest 8 short, plain names for this. No puns, no "-ly" startup names.
